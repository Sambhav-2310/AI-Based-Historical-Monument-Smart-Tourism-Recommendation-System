About the Project

Heritage AI is an academic project that uses machine learning to recommend Indian historical monuments based on a user's travel preferences.

🎯 Objective

To build an intelligent recommendation system that helps tourists discover India's rich cultural heritage by suggesting monuments aligned with their interests, budget, season, and travel duration.

🤖 ML Integration

User preferences from the form are sent to a Flask backend where a Scikit-learn model (content-based + similarity scoring) ranks monuments and returns the most relevant ones, which are then displayed on the results page.

🛠 Technologies Used

  • Python
  • Flask
  • Scikit-learn
  • HTML
  • CSS

⚙️ How It Works

1. User fills the recommendation form.
2. Inputs are sent to the Flask API.
3. The ML model computes similarity scores.
4. Top monuments are returned and rendered as cards.
